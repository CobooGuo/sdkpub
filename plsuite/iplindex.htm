Intel® Image Processing Library v2.5

Documentation

The Intel® Image Processing Library provides a set of highly optimized C functions that implement image processing functions on Intel architecture processors. You may use this library if you agree to all terms and conditions of the License Agreement.

The following documentation is available for the library:

  • Release Notes
  • Reference Manual (Acrobat* PDF format)
  • Quick Reference (Acrobat PDF format)
  • Delphi* Quick Reference (HTMLHelp and HTML formats, if installed)
  • IPLib Tutorial (if installed)
  • Readme files for examples (if installed):
    • Image Editor
    • Performance tool and data
    • UserDLL creation
    • Video effects examples

Note: The HTMLHelp documents require a browser with support for HTMLHelp.

For more information about the library, refer to the Intel Image Processing Library homepage.
