Сборка проекта VMDemoTk.

1. Разархивировать содержимое файлов dll's.zip, include&lib.zip и VMDemoTk.zip.

2. Открыв файл проекта в VC, указать каталог \include как каталог для include и для sourse files, \lib - для library files.

3. Указать путь к OpenCV \lib и \include

4. Поместить  tcl83.dll и tk83.dll, interface.tcl и iniparse.tcl в каталог exe-файла.
